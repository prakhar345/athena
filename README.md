# athena
SpringBoard Hiring Challenge Submission


# Description
A website that mocks www.coursera.org. The website provides the list of courses available at coursera for a particular query. 

# Technologies used
Front-end
HTML, CSS , JavaScript, jQuery, Bootstrap

Back-end
Django Framework

# Insturctions to run code
•	Install packages: pip install -r Requirements.txt
•	Run server: python manage.py runserver
